Computer Systems Fundamentals

This unit covers the building blocks of computer logic. With the use of 1's and 0's and logic gates, we learn the fundamentals of computer and the systems governing their functions. We would also learn how to use ASSEMBLY, a low-level programming language which is also known as machine code.


Logic Gates are used to manipulate boolean inputs to preform certain processes. What we learn in this unit are how logic gates affect boolean logics, how to simplify boolean expressions and how to build an ALU and a full-adder.


Assembly language is a low-level language that makes machine code readable. As this is the closest thing to machine code, it is useful when it comes to understanding how computer hardware functions. As a programmer, assembly is as close to the processor as you can get, making it great for optimization and having complete control over the computers' resources.



Information Systems

This unit covers the use of networks of hardware and software that people and organizations use to manipulate data. We learn the Systems Development Lifecycle(SDLC) . management information and the usage of diagrams such as the Use Case Diagram and Entity Relationship Diagrams to convey how a system functions and the various datapoints within a system.


Further along we will also use SQL to create a database for our coursework which involves designing and building a database for Ray's Rentals along with creating the relevant reports that are used to document the system's development. This course would prove useful in our field because a lot of the busy work such as documentation and presentations would be used in the work environment.
